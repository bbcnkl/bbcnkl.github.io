Face Recognition

A web application that recognizes faces based on a webcam image - in realtime. It uses machine learning algorithms from OpenCV library.
The background itself is written in the Python programming language, in which the data is processed, trained, and compared with the images in base, while the user part consists of a web application that runs on php, so this software is suitable for installation on local networks.
The project was made in course at university: Architecture of Computer Systems, with colleague Nenad Pantelic.
