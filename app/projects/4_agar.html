Agar Balkan

My hobby project of developing online games. Several games were created on multiple platforms: web, mobile, discord, facebook games.
Technologies I used:
Frontend: Canvas, Vue, Vuex, jQuery, Semantic UI, Youtube API, Facebook SDK, SEO. Gulp and webpack for workflow automatization.
Backend: PHP Codeigniter, MySQL, restfull services.
The core of the game is written in NodeJS
DevOps: nginx, apache2, server security, setting up DNS,iptables, cron.
