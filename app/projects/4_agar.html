Agar Balkan

My personal project - a mode of popular game: agar. The project consists of several parts:
Front-End application: Using Canvas, Vue, Vuex, and routing as the basis. jQuery, Ajax, Semantic UI,Youtube API, Facebook SDK, SEO. Gulp and webpack for automatization and startup.
Back-End application: PHP Codeigniter, MySQL, restfull services.
The core of the game is written in NodeJS and canvas and publicly shared, but significantly modified.
Servers: setting the lamp stack on VPS servers (apache2, mysql), setting up DNS,iptables, running the application in nodejs, php, cron.
